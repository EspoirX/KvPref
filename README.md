# KvPref

基于属性委托的 key-value 方式存储封装

可用于基本类型，对象，可基于LiveData监听状态，提供迁移数据方法。

### 使用
```gradle
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}

dependencies {
     implementation 'com.github.EspoirX:KvPref:v1.0'
}
```
